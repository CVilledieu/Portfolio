## Things that I learned

#### Keep it simple!

I started this site wanting to use a bunch of different tech. HTMX and Bootstrap were items I spent a lot of time trying to shoe horn into the project.

#### Build, destroy, and repeat

I have lost a lot of time refactoring functions and structs more than I needed to. It is part of the process though. Each time I started over from scratch I took what I just learned and was able to learn a little more.
